# Appendix A: CLI reference

The complete command-line reference for the three Tela binaries: `tela`, `telad`, and `telahubd`. Use this appendix as a lookup. The body of the book explains how and when to use each command in narrative form; this appendix lists every flag, every subcommand, and every environment variable in one place for the moments when you want to look something up rather than read about it.
