# Release process

This chapter documents how Tela releases are produced, channeled, and promoted. It is the source of truth for what users get when they ask for an update, and the reference for maintainers cutting a release. If you are a user, the [Self-update and release channels](../howto/channels.md) how-to is the practical guide; this chapter is the internal model.
