# The Tela Design Language

This chapter is the reference for the Tela Design Language: the visual language shared across all Tela products. It is design rationale only in the sense that it documents *why* the visual language looks the way it does; it is also the reference any contributor needs when building a UI that should feel like part of the Tela family. The TelaVisor chapter in the User Guide is the canonical example of TDL in practice.

A [visual component reference](https://parkscomputing.com/content/tdl-reference.html) is available that shows every TDL primitive rendered in both light and dark themes.
