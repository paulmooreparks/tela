# Appendix E: Tela Design Language

This appendix is the reference for the Tela Design Language (TDL): the visual language shared across all Tela products. Any contributor building a UI that should feel like part of the Tela family should use this as their guide. The [TelaVisor](../guide/telavisor.md) chapter is the canonical example of TDL in practice.

A [visual component reference](https://parkscomputing.com/content/tdl-reference.html) is available that shows every TDL primitive rendered in both light and dark themes.
