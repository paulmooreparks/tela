If you reached this page via tela.awansatu.net, the full path is:

When accessed through a helper tunnel, the path is:

browser (localhost:<port>) ↓ helper (TCP → WebSocket) ↓ hub (WebSocket relay) ↓ agent (WebSocket → TCP) ↓ target service

What is Tela?

Tela is a FOSS connectivity fabric that provides outbound-only tunnels, multiplexed TCP channels, zero-install client access, and browser-mediated orchestration. It is the runtime substrate for Awan Satu.
