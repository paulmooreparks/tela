Tunnel MTU

        
      

      

        

          
The WireGuard tunnel wraps each TCP segment in encryption (WireGuard), framing (WebSocket), and TLS. If the resulting packet exceeds the network path MTU, it is silently dropped.

          
The default of 1100 works in all tested environments including WSL2. Native Windows, Linux, and macOS can typically use 1280. Direct LAN connections with no proxies can use 1420. Higher values improve throughput for large transfers.

          
Both the client and agent should use the same MTU. Mismatched values can cause silent packet drops during SSH key exchange or large file transfers.
