About Tela

Tela is a free and open-source (FOSS) encrypted remote-access fabric using WireGuard tunnels. It provides secure, zero-configuration connectivity between machines across any network topology.

TelaGUI is the desktop client for Tela. It manages connections to hubs, selects machines and services, and handles CLI tools automatically.
